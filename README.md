# s21_era_practice_C
Mostly random task on C, to get a better understanding of the various parts of the language.

## s21_string+
Group project. Implementation of the string.h library with additions (with our own implementation of sprintf and sscanf functions).

## SimpleBashUtils
Developed basic Bash text utilities (cat, grep).

## s21_decimal
Group project. Implementation of the decimal.h library in the C programming language. 

## s21_matrix
Implementation of a library for processing numerical matrices in the C programming language.
